Projects

mailSift

MailSift was a hackathon project for AngelHack Silicon Valley 2015. mailSift was a web application aimed towards small businesses and startups which allowed users to link their email account to the application, allowing emails to be pre-sorted based on the tone of the message. Postive emails were given a higher score, while more negative toned messages were given a lower score. Users could also define keywords which would sort emails into catagories based on the number of times the keyword appeared in an email.
Technologies used on this project were a Django based backend and HP Haven OnDemand for the sentiment analysis of emails. This project won the HP sponser prize for best use of HP Idol OnDemand.

Spot

Spot was a hackathon project for AngelHack San Fransico 2015. Unfortunately, Spot was never finished. The goal of Spot was to create a web application which allowed users to find and connect with others to achive fitness goals. Users could set their 'home' gym, and see all other users that frequent that gym.
Technolgoies used on this project were a Django based backend and a simple HTML/CSS front end. Googe Place API was used as a method to find a list of gyms based on zip code.
